# Documentación del Portal de Rotaciones

Este directorio contiene toda la documentación del proyecto.

## Estructura

- **setup/** - Guías de instalación y configuración
- **database/** - Scripts y documentación de base de datos
- **guides/** - Guías de uso y modificación
- **troubleshooting/** - Solución de problemas

## Inicio Rápido

1. Lee [INSTALACION.md](setup/INSTALACION.md) para configurar el proyecto
2. Ejecuta los scripts SQL en [database/](database/)
3. Consulta [USUARIO.md](setup/USUARIO.md) para crear usuarios

## Soporte

Si encuentras problemas, consulta la carpeta [troubleshooting/](troubleshooting/)
